# Book API


### App Statement

An API  to find all the university books that you need.

### Feature List

- Upload books
- review books
- Get all books and display them to the user.
- Get book by specific id.
- Get reviews in each book and display them to the user.
- Delete books.
- Database Schema

## Tools & Technologies Used

### Tools 
- Swift
- Xcode
- PGAdmin
- Postgres
- Postico 2
- Postman


### Technologies 
- Vapor
- fluent
- PostgreSQL
